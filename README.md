

## About the Project 
BandMate provides a community where users can buy and sell used musical instruments and connect with other musicians 

## Stacks
Nodejs, Nestjs, Typescript, Postgres, Docker, GithubActions, TypeORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
